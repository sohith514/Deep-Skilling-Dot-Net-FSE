# ReactJS Lab 9 - Cricket App

## Objective

Demonstrate ES6 features in ReactJS.

## Features

- Display all players using map()
- Filter players with scores less than 70
- Display odd and even players using destructuring
- Merge arrays using the spread operator
- Conditional rendering using a flag variable

## Technologies Used

- ReactJS
- JavaScript ES6
- HTML
- CSS
