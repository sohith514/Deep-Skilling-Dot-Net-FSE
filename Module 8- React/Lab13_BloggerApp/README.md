# ReactJS Lab 13 - Blogger App

## Objective

Create a React application that demonstrates:
- Conditional Rendering
- Rendering Multiple Components
- List Components
- React map() Function
- Keys in React Lists

## Components

- Course Details
- Book Details
- Blog Details

## Features

- Displays Course Details
- Displays Book Details
- Displays Blog Details
- Uses React map() for rendering lists
- Uses unique keys while rendering

## Technologies Used

- ReactJS
- JavaScript (ES6)
- JSX
- CSS

## Run the Project

```bash
npm install
npm start
```
