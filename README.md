## Modules Overview

### 1. Advanced SQL
Contains advanced database management concepts and script exercises:
- SQL exercises covering advanced queries, joins, indices, and performance optimization.
- Stored Procedures development and execution examples.

### 2. Algorithms & Data Structures
Covers implementation of algorithms and data structure practices:
- Product Search System (Linear and Binary search analysis).
- Financial Forecasting (Recursive algorithms, optimization, and time complexity analyses).

### 3. Design Patterns and Principles
Demonstrations of Gang of Four (GoF) design patterns and clean code principles:
- **Singleton Pattern:** Ensuring a single class instance throughout the application life-cycle.
- **Factory Method Pattern:** Deferring instantiation to subclasses for loose coupling.

### 4. Module 04: NUnit and Moq
Unit Testing and Mocking practices using NUnit framework and Moq library:
- Testing mathematical operations (`CalcLibrary`), string parsing (`UrlHostNameParser`), collections management (`EmployeeManager`), and exchange rate feeds (`Converter`).
- Implementing parameterization, mocking dependencies, and setting up setup/teardown functions.

### 5. Module 05: Entity Framework Core 8.0
Database design and ORM mapping using EF Core:
- `RetailInventory` management system containing Data contexts, Models, Migrations, and database seeding configurations.

### 6. Module 06: ASP.NET Core 8.0 Web API
RESTful APIs design and documentation:
- Implementation of multiple Web API versions (`FirstWebAPI`, `EmployeeWebAPI`, `EmployeeCRUDAPI`).
- Configuration of Swagger for API documentation (`SwaggerDemo`).
- JWT Authentication API setup (`JWTAuthenticationAPI`).
- Integration and basic configuration of Apache Kafka servers.

### 7. Module 07: Microservices Architecture
Building scalable microservices with security and authorization:
- JWT Authentication and endpoint security (`JWTSecureAPI`).
- Role-based authorization (`JWTRoleBasedAPI`).
- Custom middleware for validating credentials and handling unauthorized access (`JWTUnauthorizedAPI`).

### 8. Module 08: React
Frontend web development using React.js:
- **Exercise1:** First React app setup and configuration.
- **Lab2:** Student Management Portal with routing components (Home, About, Contact).
- **Lab3:** Score Calculator with custom components and stylesheets.
- **Lab4:** Blog App with Post/Posts components.
- **Lab5:** Cohort Tracker with modular CSS and detailed views.
- **Lab9:** Cricket App with Indian Players list components.
- **Lab10:** Office Space Rental App with image assets.
- **Lab11:** Event Examples with Counter and Currency Converter components.
- **Lab12:** Ticket Booking App with Login/Logout, Greeting, and Guest/User pages.
- **Lab13:** Blogger App with BlogDetails, BookDetails, CourseDetails, and Data components.

### 9. Module 09: Application Debugging
Debugging React applications, error boundaries, state management, and debugging techniques:
- **Exercise 1-4:** Basic React apps (`myfirstreact`, `studentapp`, `scorecalculatorapp`, `blogapp`).
- **Exercise 5-9:** Intermediate apps (`cohorttracker`, `trainersapp`, `shoppingapp`, `counterapp`, `cricketapp`).
- **Exercise 10-13:** Advanced apps (`officespacerentalapp`, `eventexamplesapp`, `ticketbookingapp`, `bloggerapp`).
- **Exercise 14-16:** State and forms (`contextapiapp`, `formapp`, `mailregisterapp`).
- **Exercise 17-19:** API integration and error handling (`postsapp`, `calculatorapp`, `errorboundaryapp`).

### 10. Module 10: Version Control (GIT)
Hands-on exercises with Git version control:
- Repository initialization, branching strategies, merge conflicts resolution, and commit history tracing.

### 11. Module 11: DevOps and CI/CD
Implementation of continuous integration and continuous deployment pipelines:
- Pipeline configurations, automation scripts, and workflow setups.

### 12. Module 12: Containerization Using Docker
Containerizing applications for consistent environments:
- Dockerfile configurations, building images, running containers, and docker-compose configurations.

### 13. Module 13: GenAI Fundamentals
Exploring Generative AI fundamentals and integration within modern applications.
